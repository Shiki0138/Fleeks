Google Apps Script（GAS）サンプルコード

オプション1のカスタムフォームを使用する場合のGASコード


// Google Apps Script (GAS)
function doPost(e) {
  try {
    // リクエストデータを取得
    const data = JSON.parse(e.postData.contents);
    
    // スプレッドシートに記録
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    sheet.appendRow([
      new Date(),
      data.name,
      data.salon,
      data.email,
      data.phone,
      data.instagram,
      data.concern
    ]);
    
    // メール送信（申込者へ）
    const userMailBody = `
${data.name} 様

この度は、Fleeksへのお申込みありがとうございます。
以下の内容で受け付けました。

【お申込み内容】
お名前: ${data.name}
サロン名: ${data.salon || '未記入'}
メールアドレス: ${data.email}
電話番号: ${data.phone}
Instagram: ${data.instagram || '未記入'}
ご相談内容: ${data.concern}

24時間以内に担当者よりご連絡いたします。
今しばらくお待ちください。

━━━━━━━━━━━━━━━━━━
Fleeks - 美容師のためのInstagram集客マスタリー
https://www.shiki-stagram.com
━━━━━━━━━━━━━━━━━━
`;
    
    GmailApp.sendEmail(
      data.email,
      '【Fleeks】お申込みありがとうございます',
      userMailBody
    );
    
    // メール送信（管理者へ）
    const adminMailBody = `
新規申込みがありました。

【申込者情報】
日時: ${new Date()}
お名前: ${data.name}
サロン名: ${data.salon || '未記入'}
メールアドレス: ${data.email}
電話番号: ${data.phone}
Instagram: ${data.instagram || '未記入'}
ご相談内容: ${data.concern}

スプレッドシート:
https://docs.google.com/spreadsheets/d/YOUR_SPREADSHEET_ID
`;
    
    GmailApp.sendEmail(
      'admin@example.com', // 管理者メールアドレス
      '【Fleeks】新規申込み通知',
      adminMailBody
    );
    
    return ContentService.createTextOutput('Success');
    
  } catch (error) {
    console.error(error);
    return ContentService.createTextOutput('Error');
  }
}

// GETリクエスト（テスト用）
function doGet() {
  return ContentService.createTextOutput('Fleeks Form API');
}

GASの設定手順

  1. Google スプレッドシートを新規作成
  2. 拡張機能 → Apps Script を開く
  3. 上記コードを貼り付け
  4. デプロイ → 新しいデプロイ
  5. 種類: ウェブアプリ
  6. アクセスできるユーザー: 全員
  7. デプロイ後のURLをフォームのGAS_URLに設定
