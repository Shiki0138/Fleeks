# GitHub Pagesでの公開手順

## 1. GitHub リポジトリで設定

1. https://github.com/Shiki0138/Fleeks にアクセス
2. 「Settings」タブをクリック
3. 左メニューの「Pages」をクリック
4. 「Source」セクションで：
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. 「Save」をクリック

## 2. 公開URL

設定後、以下のURLでアクセスできるようになります：
https://shiki0138.github.io/Fleeks/

※反映まで数分かかる場合があります

## 3. 注意事項

- GitHub Pagesは無料で利用可能
- HTTPSが自動的に有効
- カスタムドメインも設定可能
- リポジトリがPublicである必要があります
